## My Reads

### Clone the project

`git clone https://github.com/fredygil/reactnd-project-myreads.git`

### Run the project

`cd reactnd-project-myreads`
`npm start`

### Use the app

Go to [App on localhost](http://localhost:3000). You can move books from one shelf to another by clicking on green arrow below each book. By clicking the + button on bottom right corner you can search new books.
